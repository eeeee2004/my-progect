# 酒店预订管理系统

面向中小型酒店的轻量级在线预订管理系统，支持管理员、前台、住客三类角色，覆盖房型管理、在线预订、入住退房、动态定价、统计分析和审计日志全流程。

## 快速开始

### 环境要求

- JDK 17+
- Node.js 18+
- MySQL 8.0

### 安装与运行

```bash
# 1. 克隆仓库
git clone <仓库地址>
cd rjxm

# 2. 创建数据库
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS hotel_booking DEFAULT CHARACTER SET utf8mb4"

# 3. 启动后端（默认连接 MySQL，端口8080）
cd backend
mvn spring-boot:run

# 4. 启动前端（默认端口3000）
cd frontend
npm install
npx vite
```

打开 `http://localhost:3000`，使用预置账号登录：

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | admin | 123456 |
| 前台 | front1 | 123456 |
| 住客 | guest1 | 123456 |

## 核心特性

### 用户认证与安全

- **三角色登录**：住客/前台/管理员各自独立入口，JWT 令牌鉴权。
- **登录限流**：5次密码错误后锁定15分钟，防暴力破解。
- **BCrypt加密**：所有密码密文存储，不存明文。

### 预订管理

- **在线预订**：住客浏览房型详情，选择日期和房间，填写入住信息即可预订。
- **动态定价**：房价日历支持周末溢价、节假日、促销折扣等规则，按日自动计算总价。
- **全流程追踪**：预订→入住→退房状态机，订单历史可查。

### 房型管理

- **详细信息**：8个维度描述房型（床型/面积/早餐/窗型/楼层/取消政策等）。
- **图片上传**：支持 JPG/PNG/GIF/WEBP，兼容 iPhone 和微信等多端来源。
- **独立详情页**：大图展示+信息表格+可选房间列表，可直达预订。

### 数据可视化

- **入住率折线图**：近7天/近30天趋势切换。
- **每日营收柱状图**：渐变金色柱体展示。
- **房型营收占比**：环形饼图+中心总金额。

### 审计与安全

- **操作日志**：17处关键操作自动记录（人/时间/IP/参数/结果）。
- **三级权限**：URL级别角色拦截。
- **全局异常处理**：友好的错误提示，不暴露技术细节。
